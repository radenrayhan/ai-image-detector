# 🤖 AI Image Detector — by Rayhan

Aplikasi web untuk mendeteksi apakah sebuah gambar dibuat oleh AI atau bukan.

## 🔗 Live Demo
[ai-image-detector-nu.vercel.app](https://ai-image-detector-nu.vercel.app)

## ✨ Fitur
- Upload gambar via drag & drop
- Deteksi AI vs foto asli dalam detik
- Confidence score dengan visualisasi
- Dark mode UI modern

## 🛠 Tech Stack
- Next.js 16
- Tailwind CSS
- Sightengine AI API
- Vercel (deployment)
